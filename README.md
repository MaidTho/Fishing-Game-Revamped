<h1 align="center">INSTALLATION</h1>

OPTION 1

    • Download the code files. 
    • Open folder in workspace.
    • Open GUILoginReg.py  
    • Press Run

OPTION 2

    • Copy Codespace into workspace.
    • Open GUILoginReg.py
    • Press Run.

OPTION 3 (Soon to be released)

    • Download source code .Zip file.
    • Launch .exe file "Fishing Game".
    • Enjoy!
